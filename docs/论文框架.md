# 硕士学位论文框架：基于视觉语言模型的社保表格识别

> 底座 Qwen2.5-VL-3B + ms-swift / peft，4×RTX 4090。三大研究内容串联：
> **内容1 打好复杂结构识别的模型能力 → 内容2 攻克"宽表"的长序列/视觉 token 瓶颈 → 内容3 用结构可验证信号做强化学习，低成本迁移到真实社保域。**

面向的核心难点（真实社保表）：无线表 / 复杂多级表头 / 合并单元格 / 空单元格 / 宽表。

---

## 内容1：结构分支增强 + 约束解码的复杂表格识别（核心，模型改造）

**目标**：在预训练文档 VLM 上做架构级改造，系统性提升无线表、复杂表头、合并、空单元格的结构识别质量。输入=裁剪清晰的表格图，输出=完整结构 + 单元格内容（HTML）。

### 1.1 基线（已完成）
Qwen2.5-VL-3B + LoRA，image→HTML 端到端微调。HTML 作为结构目标，`rowspan/colspan` 表示合并、空 `<td>` 保留空单元格。
v2 实测（test=526，含有线/无线）暴露问题：TEDS 0.96 / 完全匹配仅 **0.091**；有线表行数准确率 0.438、空 F1 0.477；无线表列数准确率塌到 **0.650**；合并 spanF1 两边都弱（0.66 / 0.57）。→ 高 TEDS 被平均掩盖，整表几乎不能 100% 还原，立靶明确。

### 1.2 M1 结构分支（代码已完成，待训练）
在解码器最后一层隐状态的每个 `<td` 锚点上并联轻量结构头，显式监督：合并跨度（rowspan/colspan 分类）、空/非空、（可选）占用与单元格角色。与主 HTML 分支联合训练：`L = L_html + λ_span·L_span + λ_empty·L_empty`。不改 Qwen forward，只消费 hidden states，风险低。
支撑：UniTabNet 逻辑解码器（EMNLP'24, 2409.13148）、TDATR structure-guided 模块（2026, 2603.22819）、TFLOP 逻辑结构解码器（2025, 2501.11800）。

### 1.3 M2 表头层级图（核心差异点，待实现）
对表头单元格预测父子边 + 每个数据列的表头路径（轻量图注意力/分类头）。这是全方案最具新意的一块：表头树/列表头路径的**表示**成熟（AAAI'12 表头检测分类、RealHiTBench 2506.13405、Semantic Triplet Restoration 2605.31550），但让文档 VLM 的 TSR 模型**直接输出**表头层级几乎无人做，且直击"复杂表头"痛点。评测可报 baseline 无法产出的"表头路径准确率"。

### 1.4 M3 结构↔内容双向互学习（待实现）
结构分支与 HTML 主分支共享编码、互为辅助监督，在两者结构不一致处加一致性损失，让结构与内容互相纠偏。
支撑：Multi-Cell Decoder and Mutual Learning（ICDAR'24, 2404.13268）、Hierarchical Modeling（2025, 2512.21083）。

### 1.5 C 结构约束解码（待实现）
用 M1 的占用状态在生成时屏蔽非法跨度/越界 token，把结构分支从"训练期正则"变成"推理期真在用"，闭环并直接压完全匹配率。
支撑：OTSL 结构感知词表（ICDAR'23, 2305.03393）、grammar-constrained decoding（2305.13971 等）。

### 1.6 数据
合成社保表数据集，覆盖有线/无线/复杂表头/合并/空多难度，逐图 JSON 标注（网格、单元格、跨度、角色、表头树、难度 meta），经结构合法性校验器质检。收集 + 人工校验标注。

### 1.7 评测与消融
主指标：TEDS / S-TEDS / 完全匹配率。细粒度：行数准确率、列数准确率、合并 spanF1、空单元格 F1、单元格内容准确率、**表头路径准确率**。按有线/无线、表头层数、宽表分组报告。消融：逐个去掉 M1 / M2 / M3 / C，量化各模块贡献。主对比：方案A vs 基线。

**工程落地**：走 transformers + peft 自定义训练脚本，底座 LoRA、结构头全量训。分步降险：M1 → C → M2 → M3，每步与基线对比。

## 内容2：面向宽表的视觉 token 压缩 / 长序列建模

社保宽表列数多，导致视觉 token 与输出 HTML 序列双重膨胀，显存与精度同时下滑。本内容在保结构完整的前提下，对视觉 token 做自适应压缩（或分块编码-跨块拼接建模），降低长序列开销、稳住宽表精度。承接内容1（结构识别能力已具备），专攻规模瓶颈。评测重点在宽表分组的精度-显存/时延权衡。

---

## 内容3：结构可验证的强化学习领域自适应

用内容1 的结构合法性校验器（占用无重叠/无漏覆盖、跨度合法、表头一致）作为**可验证奖励**，在少量甚至无人工标注的真实社保扫描件上做强化学习/自训练，把在合成域训好的模型迁移到真实域。承接内容1+2 的结构与规模能力，解决"真实数据标注成本高"的落地问题。评测重点在真实社保表上的结构指标与标注成本。
